# AGENTS

This repository is a Rust CLI/TUI application named `plinks`.

Key context:
- The crate version lives in `Cargo.toml`.
- Project data is stored in a checked-in `project-links.toml` file. Its config schema version is `1` and code should continue to read/write that format unless explicitly changing the schema.
- The standard verification step is `cargo test`.
- Arch packaging artifacts are generated by `./scripts/build-arch-package.sh`, which reads the package version from `Cargo.toml`.
